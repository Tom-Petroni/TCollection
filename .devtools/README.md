# TCollection Devtools

Ce dossier contient les outils de dev et de scaffolding qui servent a faire
evoluer `TCollection` et les repos de nodes, mais qui ne sont pas utiles aux
artistes dans la release runtime.

## Contenu

- `scaffold_node_repo.py`
  cree un nouveau repo node compatible `TCollection`
- `new_worktree.ps1`
  cree un worktree Git propre pour travailler sur plusieurs sujets en parallele
- `NODE_TEMPLATE_WORKFLOW_FR.md`
  documente le workflow local recommande pour les nouveaux nodes

## Important

Ces fichiers restent dans le repo source pour le dev.

Ils ne sont pas embarques dans le zip `TCollection-vX.Y.Z.zip`.
